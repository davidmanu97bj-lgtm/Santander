EXPLORA APP INSTALABLE (PWA)

Archivos incluidos:
- index.html
- manifest.json
- service-worker.js
- icons/icon-192.png
- icons/icon-512.png

Para instalar en iPhone:
1. Subí esta carpeta a GitHub Pages, Netlify, Vercel o cualquier hosting HTTPS.
2. Abrí la URL en Safari.
3. Tocá Compartir.
4. Tocá “Agregar a pantalla de inicio”.

Importante: en iPhone no se instala desde archivo local. Debe estar publicado en una URL HTTPS.
